    Selectores jQuery

    PLos selectores jQuery le permiten seleccionar y manipular elementos HTML.

    Los selectores jQuery se utilizan para "encontrar" (o seleccionar) elementos HTML en función de su nombre, id, clases, tipos, atributos, valores de atributos y mucho más. Se basa en los selectores CSS existentes y, además, tiene algunos selectores personalizados propios.

    Todos los selectores en jQuery comienzan con el signo de dólar y paréntesis: $().

    El elemento Selector

    El selector de elementos jQuery selecciona elementos en función del nombre del elemento.

    Puede seleccionar todos los elementos de una página como esta: < p >

    $("p")

    El selector de #id

    El selector jQuery utiliza el atributo id de una etiqueta HTML para encontrar el elemento específico.#id

    Un identificador debe ser único dentro de una página, por lo que debe usar el selector de #id cuando desee encontrar un solo elemento único y único.

    Para encontrar un elemento con un identificador específico, escriba un carácter hash, seguido del identificador del elemento HTML:

    $("#test")
